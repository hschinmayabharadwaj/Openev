What each policy is

Random

Trained (raw)

Heuristic

Hand-coded curriculum — the baseline ceiling for this env. Reliably reaches 0.83 task score.

Why this is different

Six things no other submission will have

Built from the env primitives up. No gridworld clone, no chess wrapper, no synthetic data farm.

01

Hybrid policy with guardrails

Trained LLM proposes; env-rule guardrails reject illegal actions before they burn turns. Every override is shown to the user — judges literally see when the model was wrong.

02

Playable cyberpunk heist

Walk an isometric city as Vex, take orders from the trained LLM ("The Operator") in your ear. WASD + E. Try doing that in a Colab notebook.

03

Transparent reasoning panel

Per-turn: prompt sent, raw model output, parsed action, latency, fallback flag. The model's mind is on screen.

04

Multi-agent reputation

4 factions with reputation deltas, alliance state, sector-specific deploy bonuses. Real coalition math, not flavor text.

05

Composable 5-gate rubric

Each step gives partial credit on alliance / resources / operation / extraction-message / extraction. Reward hacking is provably hard.

06

Race-mode SSE

Watch all 4 policies run the same task, lock-step, on one screen via Server-Sent Events. The bar chart resolves itself in 30 seconds.
