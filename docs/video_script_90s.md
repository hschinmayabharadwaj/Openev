# Neon Syndicate OpenEnv - 90 Second Demo Script

## 0:00 - 0:12 Hook
"Most LLM environments test short answers. Neon Syndicate tests strategy. In this world, the model has to negotiate with factions, manage resources, and complete a multi-step mission under uncertainty."

## 0:12 - 0:28 Problem
"Current models often fail on long-horizon planning when early mistakes compound. We built this environment to train that exact weakness: coalition reasoning plus delayed rewards in a partially observable setting."

## 0:28 - 0:50 Environment
"Each mission has stakes, rumors, and threat levels. The agent can scout sectors, negotiate pacts, trade resources, deploy assets, run an operation, and secure extraction. Success only happens if all constraints line up, so reward hacking is hard."

## 0:50 - 1:08 Training + Evidence
"We provide an end-to-end TRL training loop connected to the live environment API, and we compare policies with reward curves and success rates. You can reproduce everything from our Colab and scripts."

## 1:08 - 1:25 Outcome
"After training, we see better coalition choices, fewer invalid action sequences, and improved mission completion scores. This is exactly the behavior we want for real multi-agent workflows."

## 1:25 - 1:30 Close
"Neon Syndicate pushes LLMs beyond next-token prediction into strategic world modeling."
